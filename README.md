# 🌍 Global Weather Data Analysis Dashboard

## 📌 Project Overview

The Global Weather Data Analysis Dashboard is a data analytics project
developed using Excel and Power BI to clean, analyze, and visualize
weather data.

This project transforms raw weather records into meaningful insights
using interactive dashboards and visual reports.

------------------------------------------------------------------------

## 🎯 Objectives

-   Clean and preprocess weather data
-   Analyze weather patterns
-   Build an interactive dashboard
-   Generate meaningful insights

------------------------------------------------------------------------

## 🛠 Tools & Technologies

-   Microsoft Excel
-   Power BI Desktop
-   CSV Dataset
-   Data Visualization
